Carray is a compound variable where each event time in the box is saved with an event code: .1 = Right lever press .2 = Left Lever press .3 = Nosepoke .4 = FREE PELLET .5 = EARNED PELLET The number BEFORE the decimal is a time stamp. The two rightmost numbers are .XY For example, 1800 would be 18.00 seconds.

So now we should decompose them into different variables: Time and Event.

# Separate the 'Carray' column into 'Time' and 'Event.'
